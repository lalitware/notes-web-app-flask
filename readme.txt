Aim : The aim is to create a notes web app using flask where user can sign up and notes.
      The notes will be stored in the database.
Begin with some package installation
1. pip install flask
2. pip install flask-login
3. pip install flask-sqlalchemy

Tutorial Url: https://www.youtube.com/watch?v=dam0GPOAvVI&t=2448s

To run the app:
a. select the select the interpretor
b. run /home/*username*/anaconda3/bin/python /var/www/html/flask-notes-web-app/main.py
